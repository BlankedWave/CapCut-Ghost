# 🎬 CapCut Ghost

---

## 🧠 How It Works (The Logic)

1. The `.bat` file blocks CapCut's auto-updater so it stays on an older version (4.0.0).
2. The `.exe` acts as a simple **toggle switch**:
   - Type `on` → Triggers a pro mode export (enables you to export video with pro features).
   - Type `off` → Resets the flag (so Pro features reappear for editing).
3. You export your project while the toggle is `on`, and edit while it is `off`.

## 📋 Step-by-Step Instructions

### Step 1 – Install CapCut
- Run the downloaded `.exe` and install CapCut v4.0.0.
- **Do not** update the software during installation.

---

### Step 2 – Initialize the App
- Open CapCut.
- Let it load completely, then **close** the application.

---

### Step 3 – Block Auto-Updates
- Navigate to the `Assets` folder.
- Right-click **`CapCut Update Stopper.bat`** and select **"Run as administrator"**.
- Wait for the confirmation message that updates have been blocked.

---

### Step 4 – Edit Your Video (Normal Mode)
- Open CapCut.
- Edit your video using the available tools.
- **Exit and save** your project.

---

### Step 5 – Activate Toggle (Export Mode)
- Go to the `Assets` folder and run **`CapcutCrack.exe`**.
- In the terminal, type: on
- Press **Enter** and wait for CapCut to restart automatically.

---

### Step 6 – Export Your Project
- Once CapCut reopens, open your saved project.
- **Export** your video.

---

### Step 7 – Deactivate Toggle (Edit Mode)
- Close CapCut.
- Run **`CapcutCrack.exe`** again.
- In the terminal, type: off
- Press **Enter**.
- Reopen CapCut to continue editing with Pro features visible.

---

### Step 8 – Repeat for Other Projects
- For each new project, repeat **Steps 4 through 7**:
- Edit → Save → Toggle `on` → Export → Toggle `off`
